## 星河环球大学（GGU）简介

**星河环球大学 Galaxy Global University**是一所以星际科技、全球创新与未来工程为核心特色的国际化研究型大学。学校立足星河视野，面向全球发展，汇聚世界前沿科技力量与优质教育资源，致力于培养具备国际视野、创新精神、工程能力与社会担当的新时代领军人才。

GGU以高端理工与前沿交叉学科为优势方向，重点布局深空科技、人工智能、智能制造、未来医学与全球治理等战略领域，构建融合科研、产业与国际合作的创新生态体系，打造面向未来的世界一流高等学府。

学校秉承“探索未知，追求卓越；立足科技，服务全球；兼容并包，创新引领”的办学理念，坚持开放办学与全球协作，构建国际化课程体系与科研平台，推动学生全程参与科研实践与国际交流。

**星河为梦，环球为路。**  
GGU以科技为翼，以创新为魂，与世界同行，与未来同在。
